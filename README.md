# varseldb-api

API for å opprette varsler som vises i dekoratøren ved innlogget bruker. Midlertidig løsning frem til alle varslinger går via brukernotifikasjoner.

# Kom i gang
1. Bygg varseldb-api ved å kjøre `gradle build`
